## 🚀 它是干嘛的？ / What is this?

简单来说，这是我的个人自留地。我在这里：
*   瞎折腾一些前端组件和好看的 UI（努力向硬核、工业风靠拢，希望没翻车）。
*   部署一些自己写的后端小玩具（只要它不崩溃，那就是成功的部署）。
*   记录我的学习笔记、踩坑记录，或者单纯存放一些好玩的东西。

Simply put, this is my personal sandbox. Here I:
*   Mess around with front-end components and cool UIs (aiming for that sleek, industrial/tech vibe, hope I nailed it).
*   Deploy some homegrown back-end toys (if it doesn't crash, it's a successful deployment).
*   Log my learning journey, debug notes, or just store things I find interesting.
